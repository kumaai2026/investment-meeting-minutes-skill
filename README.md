# 投资会议纪要整理 Skill

这个仓库保存 Codex 使用的中文投资会议纪要整理 skill 包。

## 内容

- `skills/投资会议纪要整理`：基础 skill，包含归档、转录、校对、导出、Dify 复核链路和验证脚本。
- `skills/投资会议纪要-多人复盘会`：多人复盘会类型 skill。
- `skills/投资会议纪要-上市公司交流`：上市公司交流类型 skill。
- `skills/投资会议纪要-专家交流`：专家交流类型 skill。
- `skills/投资会议纪要-其他`：其他或自定义会议类型 skill。

## 注意

- skill 文件必须保持 UTF-8 无 BOM 和 LF 换行。
- 本仓库不包含本机私有配置、API key、运行缓存、模型权重或虚拟环境。
- 脚本中的本机路径是当前部署默认值，迁移到其他机器时需要映射到对应环境。
